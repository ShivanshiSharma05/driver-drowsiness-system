# 🚗 AI-Powered Driver Safety Monitoring System

An intelligent real-time driver monitoring system built using **Python, OpenCV, Flask, and Computer Vision** to detect driver drowsiness and improve road safety.

The system continuously monitors the driver's face and eye activity using a webcam feed and generates alerts when signs of fatigue or drowsiness are detected.

---

# ✨ Features

* 🔴 Real-time driver monitoring
* 👁️ Eye detection and tracking
* 😴 Drowsiness detection system
* 🚨 Audio alert mechanism
* 🌐 Flask-based live dashboard
* 📊 Event logging using CSV
* 🎥 Live webcam streaming
* 🧠 Computer Vision powered monitoring
* 💻 Interactive and modern UI
